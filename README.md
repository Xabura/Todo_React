This is a simple Todo Application which is connected to database using mockapi.io and different tools.

tools used:
React Hooks
Custome React Hooks
React Router
API (mockapi.io)
LocalStorage
Responsive UI/UX
Refined file structure
Css pre processor SCSS
Dark/Light themes

How to start the project:
clone repository
run: "npm install"
run: "npm run dev"
